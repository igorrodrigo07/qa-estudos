# QA Estudos

Repositório dedicado aos meus estudos em Quality Assurance (QA), com foco na certificação CTFL (Certified Tester Foundation Level).

## Objetivo

Consolidar conhecimentos fundamentais de testes de software, incluindo:

- Fundamentos de Teste
- Ciclo de Vida de Desenvolvimento de Software (SDLC)
- Técnicas de Teste
- Níveis e Tipos de Teste
- Gestão de Testes
- Ferramentas de apoio

## Estrutura

Este repositório contém:

- Resumos teóricos organizados por tópico
- Anotações práticas
- Exemplos aplicados
- Materiais de revisão para certificação

## Propósito

Este material faz parte da minha transição para a área de Quality Assurance, demonstrando organização, disciplina e evolução contínua na área de testes de software.
